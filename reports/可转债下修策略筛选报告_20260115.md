# 可转债下修策略筛选报告

**报告日期**: 2026年1月15日

## 一、筛选策略概述

### 核心逻辑
有实际业务 + 略有盈利 + 转债偿债压力大 + 有下修历史 → 大概率继续下修促转股

### 筛选条件
- **盈利组**: 存续 + 有营收(>5亿) + 盈利 + 规模/净利压力大 + 有下修历史
- **亏损组**: 存续 + 亏损 + 营收>10亿 + 有下修历史（困境反转潜力）

### 价格分档
- **强烈关注**: < 110元
- **重点关注**: 110-120元
- **次重点关注**: 120-130元
- **一般关注**: 130-140元
- **边缘关注**: 140-150元
- **观望**: ≥150元

## 五、风险提示

1. **下修不确定性**: 下修需股东大会通过，存在否决风险
2. **正股基本面**: 亏损公司存在持续恶化可能
3. **流动性风险**: 部分小规模转债流动性较差
4. **强赎风险**: 强赎进度>100%的标的可能随时公告强赎
5. **市场风险**: 整体市场下跌可能拖累转债价格
